Position Lecture

Position Static

The default value for the position property. Top, bottom, left, and right will have NO effect.

Position Relative

The element is positioned relative to its normal position. Top, bottom, left, and right have effect.

Position Absolute

The element is positioned relative to its closest positioned ancestor. Top, bottom, left, and right have effect. When you have an absolutely positioned element, it's almost always a good idea to relatively position its parent.

Position Fixed

The element is positioned relative to the browser window. Top, bottom, left, and right have effect.

Position Sticky

The element is positioned relative to the user's scroll position. Set the top property.
